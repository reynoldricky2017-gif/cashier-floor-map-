# Cashier Floor Map

A fully functional, interactive restaurant cashier floor map interface built with vanilla HTML5, CSS3, and JavaScript.

## Features

- **20-Table Restaurant Layout**: Realistic, non-grid arrangement with varied positioning
- **Color-Coded Table Statuses**: 
  - 🟢 Empty (Green)
  - 🟡 Seated (Amber)
  - 🔴 Ordering (Red)
  - 🟣 Preparing (Indigo)
  - 🟣 Ready (Purple)
  - ⚪ Paid (Gray)

- **Fully Interactive Buttons**:
  - Floor detail actions (Send check, Reassign, Mark ready)
  - Alert panel management (Assign to tables)
  - Order panel controls (Confirm, Adjust, Mark ready, Call guest)
  - Host stand operations (Seat parties at tables)

- **Dynamic View Switching**: Floor map, alerts, orders, and host stand views
- **Real-Time Feedback**: Emoji-enhanced status messages for all actions
- **Touch-Friendly Design**: Minimum 48px buttons for easy interaction
- **Single-File Application**: No external dependencies or build process needed

## Quick Start

1. Open `index.html` in your web browser
2. Click on tables to select them and view details
3. Use the view buttons to switch between floor map, alerts, orders, and host stand
4. Click any action button to update table status and see live feedback

## Technologies

- HTML5
- CSS3 (with CSS variables for theming)
- Vanilla JavaScript (no frameworks)

## Project Structure

- `index.html` - Complete application (styles and scripts embedded)

## Usage

All buttons on the page are fully functional:
- Select a table by clicking it
- Use floor actions to manage table service
- Assign waiting parties to available tables
- Manage orders and payment status
- All actions provide immediate visual feedback

Works in all modern browsers (Chrome, Firefox, Safari, Edge)
